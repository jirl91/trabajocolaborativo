# Proyecto Parquímetro

Este es un proyecto de parquímetro desarrollado en Java con interfaz gráfica mediante Java Swing. Permite gestionar el tiempo de estacionamiento de un vehículo y realizar el pago correspondiente a través de diversos medios, como efectivo o tarjeta.

## Descripción

El proyecto Parquímetro es una aplicación de escritorio que simula un sistema de parquímetro. Permite a los usuarios registrar la hora de entrada y salida de un vehículo, ingresar datos del cliente y del vehículo, calcular el tiempo de estacionamiento y el costo asociado, y realizar el pago mediante diferentes medios.

## Características

- Registro de la hora de entrada y salida.
- Ingreso de datos del cliente y del vehículo.
- Cálculo automático del tiempo de estacionamiento y costo.
- Pago con efectivo y tarjetas.
- Interfaz gráfica amigable.

## Requisitos

- JDK instalado (versión X o superior).
- IDE compatible con Java (Eclipse, IntelliJ, etc.).
- Conexión a Internet para descargar dependencias (si es necesario).
